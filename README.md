# 🛒 Product App (Flutter E-Commerce)

A simple and clean Flutter-based E-Commerce application with product listing, wishlist, search, and filtering features. Built using Riverpod and Hive for state management and local storage.

---

## 🚀 Features

- 📦 Product Listing (Grid UI)
- 🔍 Search Products
- 🏷️ Category Filter
- 💰 Price Range Filter
- ❤️ Wishlist (Add/Remove)
- 💾 Local Storage using Hive
- 📄 Product Detail Screen
- 🔗 Share Product Feature
- 🎨 Smooth UI with Hero Animations
- 📱 Bottom Navigation (Home, Wishlist, Profile)

---

## 🧰 Tech Stack

- Flutter
- Dart
- Riverpod (State Management)
- Hive (Local Database)
- Cached Network Image
- Share Plus

---

## 📱 Screens

- Home Screen (Product Grid)
